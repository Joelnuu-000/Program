一、 研究動機：現行生態數位化之瓶頸反思

探討現行數位圖鑑流於單向資訊灌輸，缺乏互動倫理引引與人性化對話溫度的現況。

下載提案文件 (DOCX) (NOTEBOOKLM)

1. 單向度資訊灌輸之局限
  • 現行數位生態系統多偏重於靜態生物分類學資料（如界門綱目科屬種、型態特徵文字）的展示。
  • 此類結構化資料缺乏情境脈絡與互動趣味，易導致學習者產生感官疏離，難以深化環境倫理。
  • 數位化流於形式，未能發揮科技增強學習（Technology-Enhanced Learning）的雙向反饋優勢。
2. 現場專業導覽人力供需失衡
  • 生態復育基地（如苗栗獨角仙復育現場）具備高度季節性，專業導覽人才（如台大昆蟲系專家）在高峰期運能嚴重不足。
  • 缺乏即時互動引導與環境倫理教育，易導致遊客因缺乏認知而產生不當採集行為，對在地復育造成二度生態破壞。
  • 急需一套能數位孿生專家領域知識、兼具人性化交談風格的生成式 Agent 系統。

二、 技術痛點：戶外極端環境之視覺辨識干擾

探討泛用型物件偵測模型於複雜田野環境下，因環境光源與生物行為導致的辨識失準。

環境背景
干擾
【 環境雜訊與輪廓不全 】 戶外光臘樹林在夏季高強度日照下，易產生大逆光與破碎陰影干擾。生物光滑鞘翅亦會引發高亮度反光；加上昆蟲在野外具備隱匿、葉片遮蔽、或交配打架等重疊行為，導致傳統 YOLO 全局邊界框因特徵不全而引發漏看。
層級化
特徵救援
【 區域注意力機制與部件特徵對齊 】 本計畫於資料集標註階段導入「層級化部件標註」，細分生物局部特徵（如觭角、大顎、前胸背板）。當全局信心度因遮蔽而低於臨界值時，後端系統將強行觸發空間注意力機制，藉由高信心度的局部特徵進行反向推理救援，確保系統之強健性（Robustness）。

三、 知識工程：基於田野訪談之專家職能與人格蒸餾（Skill & Persona Distillation）

本專題核心在於將專家難以言傳的「暗默知識」與個人特質，系統化地提煉並封裝進 AI 模型中。

STAGE 01

暗默知識擷取 (Tacit Knowledge)

針對台大昆蟲系專業人才進行深度田野跟拍與訪談。採集其在北橫、苗栗等地多年累積的「非結構化」生態觀察筆記、微棲地行為數據與解說心法。

STAGE 02

人格特徵工程 (Persona Profiling)

透過語音轉錄逐字稿與行為分析，精確提煉專家在現場導覽時的「高能量語速、科普轉化技巧、與幽默吐槽的互動邏輯框架」，將個人特質進行特徵化編碼。

STAGE 03

職能封裝與對齊 (Skill Alignment)

將提煉出的專家大腦（RAG正確數據）與專家靈魂（人格提示詞工程）進行深度整合，封裝至本地邊緣端小模型（Ollama），完美達成專家職能的數位孿生。

💡 專題核心價值： 透過「職能與人格蒸餾技術」，本系統不只能在完全斷網的復育基地提供毫秒級的極速交談響應，更打破了傳統數位圖鑑缺乏人性化引導的缺點，將最寶貴的專家解說溫度 100% 在科技載體中實現。

四、 系統架構與資料流運行圖 (Runtime Data Flow)

本專題各技術元件於系統執行期間（Runtime）的同步調度與整合邏輯：

1. 前端請求 終端載具網頁（React/Vue Web-AR）或室內全息箱捕捉昆蟲影像 ──► 透過 API 傳輸至 Node.js
2. 視覺推斷 Node.js 管家轉發 ──► 本地端 Python FastAPI 驅動 YOLO v11 執行細粒度部件辨識救援
3. 數據對齊 YOLO 輸出物種分類標籤 ──► 觸發 ChromaDB (RAG 向量知識庫) 檢索，100% 擷取正確專家田調文本塊
4. 生成響應 經蒸餾封裝之專家文獻與人格化提示詞融合 ──► 餵給 Ollama 地端模型 ──► 同步生成專家語音與動態動作 JSON 指令
🏁 5. 前端渲染 ──► Node.js 分發回傳 ──► 觸發 Unity 3D 角色與行動載具同步進行流暢的專家導覽與互動演出。

七、 研究方法與執行時程

從田野訪談、資料標註、模型蒸餾，到現場展示與回饋收斂，形成完整的研究與實驗流程。

研究方法

  • 蒐集戶外生態場景影像與專家訪談逐字稿，建立多模態資料庫。
  • 透過分層標註與部件特徵對齊，提高模型在逆光與遮蔽情況下的辨識穩定性。
  • 結合 RAG 知識檢索與人格提示詞工程，讓系統具備專家式解說與溫度化互動。
  • 於實地展示場域進行試用與回饋收斂，持續優化介面與互動流程。

執行時程

Q1資料蒐集與田野觀察，建立初步影像與訪談資料集。
Q2完成層級化標註與模型訓練，驗證辨識救援機制。
Q3整合知識庫、語音生成與互動介面，完成原型系統雛形。
Q4進行現場展示測試，收斂使用者回饋並進行成果整理與推廣。
結語： 此專題不僅是技術驗證，更是連結生態教育、數位人文與尖端 AI 的整體實踐藍圖。
